# 图的遍历

## 题目描述

给出$N$个点，$M$条边的有向图，对于每个点$v$，求$A(v)$表示从点$v$出发，能到达的编号最大的点。

## 输入格式

第1 行，2 个整数$N,M$。

接下来$M$行，每行2个整数$U_i,V_i$，表示边$(U_i,V_i)$。点用$1, 2,\cdots,N$编号。

## 输出格式

N 个整数$A(1),A(2),\cdots,A(N)$。

## 样例 #1

### 样例输入 #1

```
4 3
1 2
2 4
4 3
```

### 样例输出 #1

```
4 4 3 4
```

## 提示

• 对于60% 的数据，$1 \le N . M \le 10^3$；

• 对于100% 的数据，$1 \le N , M \le 10^5$。
